# ⚡ NovAds — Freelance Advertising Portfolio

A personal freelance advertising website showcasing 40 premium brand campaigns, built with HTML, CSS & Vanilla JS. Deployed on GitHub Pages.

## Overview

NovAds is a cinematic freelance advertising agency portfolio featuring:

- **40 brand campaigns** — from Nike, Coca-Cola and Tesla to emerging local brands
- **3D Hero Ad Showcase** — a jaw-dropping Nike campaign card with CSS 3D transforms and mouse-tracking tilt
- **7 in-depth portfolio case studies** with 3D flip cards and modal case studies
- **portfolio_2 showcase card** — a live GitHub Pages project demonstrating real full-stack work
- **Services section** with pricing
- **Testimonials carousel**
- **Contact form** (Formspree-ready)

## Tech Stack

- **HTML5** — semantic, accessible markup
- **CSS3** — custom properties, CSS Grid, Flexbox, `@keyframes`, `transform-style: preserve-3d`
- **Vanilla JS** — zero frameworks, zero dependencies
- **Google Fonts** — Inter + Syne

## Features

- ⚡ Pure CSS 3D card animations with JS mouse-tracking tilt
- 🎨 Animated gradient borders on the portfolio_2 featured card
- 📱 Fully responsive — mobile-first, touch-device aware
- ♿ Accessible — ARIA labels, focus-visible, reduced-motion support
- 🚀 GitHub Pages deployable — no build step required
